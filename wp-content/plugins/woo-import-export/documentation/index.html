A) Introduction - top

Woo Import Export Plugin is an easy, quick and advanced Import & Export your store data.

Woo Import Export Plugin gives you ability to export you store data into CSV file and import from CSV file. All type of your WooCommerce Products, Orders, Users, Porduct Categories and Coupons import/export in just one click. A great way to manage Products, Orders, Product Categories, Users and Coupons data between multiple sites.

Woo Import Export plugins inbult functionality are Multiple Powerful Filters, Export Management, Field Management, Scheduled Management.

B) Installation - top

Woo Import Export Plugin requires WordPress 3.7+

Woo Import Export Plugin has been successfully tested with WooCommerce 2.1.x, 2.2.x, 2.3.x, 2.4.x., 2.5.x

Just like every plugins, just copy the woo-import-export folder in your wp-content/plugins folder then activate it through WordPress dashboard. The plugin is only accessible for Admin's

If the installation went OK, you can now access the plugin Dashboard from the 'Woo Imp Exp' menu.

You should see four sub menu in page. Just click it for what you want.

F) Exporting Products - top

All filter that you see in page are optional. You can export all your product data in one big CSV file. You can try it by simply click on the "Export" button or click on "Preview" button.

Exporting Product options

a) Filter By Product Category

In Your store multiple category may be available but if you want to export specific category then this plugin can allow you to do that simple way by just selecting category for list of all categories.

Default : All categories, if you leave it blank.

b) Filter By Product ID / Name

This Plugin allow you to export product by only spicefic id or name. multiple product selection available so you can selelct multiple product for filter.

Default : All Products, if you leave it blank.

c) Filter By Product Author Name / Email

Product author can not be same for all products, so if you want to seelect specifilc product author than this filter is for you. Just select author from author list and click for preview or export.

Default : All authors, if you leave it blank.

d) Limit Records

In one store thousands of records are available but if you want to select only limited recored than this filter help you. this can be variable for variable product.

Default : All Records, if you leave it blank.

e) Offset Records

If in you store large number of records are available but if you export one by one page than this filter is used. by means this filter is used for paging.if you want to get records after XX(any number) records then this will help you.

Default : 0, if you leave it blank.

f) Filter By Date

If you want to export product data only for a specific period of time, you can specify start and end date using the two dedicated field. To select the desired date just click on the field to popup the calendar and select one day. You can't select a day in the future.

Default : All duration, if you leave it blank.

e) Scheduled Export

If you want to export products data Recursively than just click on 'Scheduled Expor' checkbox. After This process two option available there, first one is for interval time and other is for send mail to any recipients.

For save Scheduled just click on 'Save Scheduled' button. For manage Scheduled just click on 'manage Scheduled' menu.

K) Importing Products - top

Importing tools is the default operation you can do with Woo Import Export Plugin. This allow you to fetch all data from csv file and create, skip or update product if already exist in your store.

This feature allows to you for update your store or change your store from local to live or any other. In any situatin if you transfer store product data from one site to another this will capable fot it.

Important Note :
Imported product can not be Revoked and any reverse operation are not available after import complete, so first backup your store data before you import any data.

Importing Product options

a) Select File to Import

The file for imported product, only csv file is allowed and only this plugin exported file can be allowed for Import Product. Any other file can be Harmful for your store.

If you problem with upload file so you can change you file upload limit from server, Or you can First export Limited records and import one by one.

b) Enter URL to Import

If you file is online then only file url through you can import product. just enter URL for file.

File is copy via php copy or CURL function so check permition for file. if file dose not exist than no operation can be performed. So please check File currect URL and then click import file.

c) Select Category for Product

If you wat to create or update product with new category by importing csv, Than select category for all product that are in csv file.

Default : original category in csv, if you leave it blank.

d) Product Create / Update / Skip

If Product already available in you store than what can do? this allow you to skip, create or update product if product already available in your store. if product is not found in your store than it will be created.

There are option available for skip, update or create product select one option that is best for your store.

Product duplication is found on basis of SKU. so sku must be unique in your store. Empty SKU can not be consider.
